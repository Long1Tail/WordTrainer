# WordTrainer

Это  простенький проект, написанный на PyQT5. Его цель - предоставить возможность изучать слова.

---

# Как запустить?

Всё достаточно просто. К несчастью, я пока не разобрался, как нормально сделать релиз, но вот цепочка комманд для запуска приложений:

## 1. Скачай проект

```
git clone https://github.com/Long1Tail/WordTrainer
```

## 2. Создай и активируй виртуальное окружение

```
python -m venv .venv
.\.venv\Scripts\activate
```

## 3. Установи всё требуемое для проекта

```
pip install -r requirements.txt
```

## 4. Запусти

```
python wordTrainer.py
```
